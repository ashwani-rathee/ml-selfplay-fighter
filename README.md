## Self-Play Fighter - [Video](https://youtu.be/9t4YBXZe7hc)

[https://forum.unity.com/threads/self-play-with-automatic-curriculum.921923/](https://forum.unity.com/threads/self-play-with-automatic-curriculum.921923/)
